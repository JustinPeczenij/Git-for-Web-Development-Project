        Fork repository
        git clone <link> to clone the repo to the directly I'm in
        cd into the new repo
        git checkout -b first-last to create my own branch name for the repo
        code . to open Visual Studio
        git add . after making changes to make them ready for commiting
        git commit -m 'commit message' will commit the changes with my commit message
        git push -u origin branch-name (only use branch name once) will push all commits into local machine up into github
        now that changes are in github, I can submit a pull request
